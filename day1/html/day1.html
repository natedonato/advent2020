Example solution for part 1 using Node.js :

First we read our input from a text file, then we split our input string into a more workable form: an array of integers.  Next using a simple nested loop we can check each integer against every other integer, until we find the pair that sums to 2020, and finally we print the product of that pair to the console.
        const fs = require('fs');
const input = fs.readFileSync('input.txt', "utf8").split('\n').map(el => parseInt(el));

input.forEach((num, idx) => {
  input.forEach((num2, idx2) => {
    if (num + num2 === 2020 && idx !== idx2) {
      console.log(num * num2);
    }
  })
})

Example solution for part 2 using Node.js :

For part 2 we can use nearly the same strategy as the previous answer, but instead with another nested loop to add a third integer.
          const fs = require('fs');
const input = fs.readFileSync('day1/node/input.txt', "utf8").split('\n').map(el => parseInt(el));

input.forEach((num1, idx1) => {
    input.forEach((num2, idx2) => {
        input.forEach((num3, idx3) => {
            if(num1 + num2 + num3 === 2020 && (idx1 !== idx2 && idx2 !== idx3 && idx1 !== idx3)){
                console.log(num1*num2*num3)
            }
        })
    })
})
